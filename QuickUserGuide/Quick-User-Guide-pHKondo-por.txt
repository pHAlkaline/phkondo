pHKondo Guia Rápido

Adicionar condomínio (s) -> Criar Condomínio
Ver detalhes do condomínio

Entrar nas Frações
Adicionar propriedades -> Criar Fração
Ver detalhes da fração

Dentro da Fracção Entrar nos Proprietários
Adicionar proprietários -> Criar Proprietário
Ou use Adicionar Proprietário para pesquisar proprietários já registados na sua base de dados

Regresse ao detalhes do condomínio
Entar nos Administradores
Adicionar administradores -> Adicionar Administrador
* Administrador é o proprietário(s) selecionado que serve de elo de ligação/representante entre a gestão do Condomínio/empresa/serviço e todos os proprietários.

Regresse ao detalhes do condomínio
Entrar nos Exercícios
Adicionar exercício -> Criar exercício
* Exercício/Ano Fiscal é usado para configurar períodos de gestão, pode o período que entender, não necessariamente um ano.
É usado para controlar e agrupar dados em:
- Saldos de contas, Orçamentos...
- Relatórios para Balancete, receitas, despesas, em dívida, mapa de quotas
* IMPORTANTE: Atenção, os dados apresentados são agrupados conforme o exercício ativo, para ver dados de outro execicio este deve ser ativado posteriormente.

Regresse ao detalhes do condomínio
Entrar nas contas
Adicionar conta -> Nova Conta
Ver detalhes da conta

Dentro da Conta Entrar nos Movimentos
Adicionar movimentos de conta -> Criar Movimento
* Cada movimento tem de pertencer ao exercício ativo
* Para cada exercício é necessário criar o primeiro movimento ( no inicio do ano fiscal - abertura ) e o último movimento ( no final do ano fiscal - fecho ) , isso permitirá que o pHKondo possa realizar os relatórios baseados nos intervalos ( inicio / fim ) do exercício.
* O primeiro movimento deve ser:
O montante inicial no balanço de abertura no ano fiscal
Escolha Abertura / Fecho no campo Categoria
Escolha Abertura Balancete no campo Operação Movimento
Tipo de movimento: Crédito se saldo de conta no inicio do exercicio é 0 ou superior, Débito se saldo de conta no inicio do exercicio é < 0

* O último movimento:
Escolha Abertura / Encerramento no campo Categoria
Escolha Fecho Balancete no campo Operação Movimento

Regresse ao detalhes do condomínio
Entrar nos Orçamentos
Adicionar orçamentos -> Criar Orçamento
Defina os seus orçamentos.
Defina bem os campos de Valor, Fundo de Reserva (%), Data de Início, Quotas, Periodicidade, Distribuição, Prazo de pagamento. Vai possibilitar criar notas de débito de forma automática.
Para notas de débito automático entre em Criar Notas, atenção que o orçamento deve estar no estado ativo.
